 Descrição do Projeto

Este repositório contém os arquivos referentes ao trabalho final da disciplina. O grupo realizou o desenvolvimento colaborativo utilizando Git e GitHub, seguindo práticas de versionamento e controle de alterações.

 Etapas Realizadas pelo Grupo
1. Criação do Repositório

O repositório foi criado por João Nagata diretamente no GitHub.

2. Clonagem do Repositório

João realizou a clonagem utilizando o comando:

git clone git@github.com:JoaoNagata/Trabalho_final-.git

3. Primeiro Arquivo Criado

João criou o arquivo algoritmo_exemplo.por.txt e enviou ao repositório usando:

git add .
git commit -m "Adicionei o arquivo algoritmo.por"
git push

4. Contribuição do Segundo Computador

Maria acessou o repositório em outro computador, criou o arquivo Contribuicao2.txt e enviou:

git add .
git commit -m "Edição feita no computador pessoal"
git push

5. Sincronização das Máquinas

João atualizou seu repositório local com:

git pull

6. Criação e Edição do README

O arquivo README foi escrito por João Nagata e enviado utilizando:

git add README.md
git commit -m "Adicionado README"
git push

 Quem Fez o Quê
